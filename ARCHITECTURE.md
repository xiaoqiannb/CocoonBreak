## 数据流图

### 推荐内容获取流程

```
用户点击"获取推荐内容"
        │
        ▼
前端发送 Tauri 命令
        │
        ▼
fetch_recommendations 命令处理
        │
        ▼
BilibiliClient.fetch_recommendations()
        │
        ▼
HeadlessBrowser.navigate_to_url("https://www.bilibili.com")
        │
        ▼
获取页面 HTML 内容
        │
        ▼
使用 scraper 解析视频信息
        │
        ▼
返回 Video[] 给前端
        │
        ▼
前端显示视频列表
```

### 内容分析流程

```
用户点击"分析内容"
        │
        ▼
前端发送视频数据和 API Key
        │
        ▼
analyze_content 命令处理
        │
        ▼
LLMClient.analyze_content(videos)
        │
        ▼
构建 LLM 提示词
        │
        ▼
发送请求到 OpenAI API
        │
        ▼
接收并解析 LLM 响应
        │
        ▼
返回 ContentAnalysis 给前端
        │
        ▼
前端显示分析结果
```

### 反向搜索生成流程

```
用户点击"生成反向搜索"
        │
        ▼
前端发送内容分析数据
        │
        ▼
generate_reverse_search 命令处理
        │
        ▼
LLMClient.generate_reverse_search(analysis)
        │
        ▼
基于分析结果构建提示词
        │
        ▼
请求 LLM 生成反向搜索策略
        │
        ▼
返回 ReverseSearch 给前端
        │
        ▼
前端显示搜索策略
```

### 破解茧房流程

```
用户点击"开始破解茧房"
        │
        ▼
前端发送搜索词列表
        │
        ▼
search_and_watch 命令处理
        │
        ▼
遍历每个搜索词
        │
        ▼
BilibiliClient.search_videos(term)
        │
        ▼
发送进度更新到前端
        │
        ▼
选择第一个视频
        │
        ▼
BilibiliClient.watch_video(url)
        │
        ▼
等待30秒（模拟观看）
        │
        ▼
继续下一个搜索词
        │
        ▼
所有搜索词处理完成
        │
        ▼
发送完成通知到前端
```

## 模块说明

### 前端模块

#### [index.html](dist/index.html)
- 应用主界面
- 包含 API 配置、控制按钮、内容展示区域

#### [styles.css](dist/styles.css)
- 现代化 UI 样式
- 响应式设计
- 渐变色主题

#### [app.js](dist/app.js)
- 前端交互逻辑
- Tauri 命令调用
- 事件处理和状态管理

### 后端模块

#### [main.rs](src/main.rs)
- 应用入口点
- 初始化 Tauri 应用

#### [lib.rs](src/lib.rs)
- Tauri 应用配置
- 命令注册
- 状态管理

#### [models.rs](src/models.rs)
- 数据结构定义
- Video, ContentAnalysis, ReverseSearch, SearchProgress

#### [browser.rs](src/browser.rs)
- Chromium 无头浏览器封装
- 页面导航和内容获取
- JavaScript 执行

#### [bilibili.rs](src/bilibili.rs)
- Bilibili 网站交互
- 推荐内容抓取
- 视频搜索和观看

#### [llm.rs](src/llm.rs)
- LLM API 集成
- 内容分析
- 反向搜索生成

#### [commands.rs](src/commands.rs)
- Tauri 命令实现
- 前后端通信
- 业务逻辑协调

## 技术栈详情

### 后端技术

- **Rust**: 系统编程语言，提供高性能和内存安全
- **Tauri 2**: 跨平台桌面应用框架
- **Tokio**: 异步运行时
- **headless_chrome**: Chromium 无头浏览器控制
- **scraper**: HTML 解析
- **reqwest**: HTTP 客户端
- **serde/serde_json**: 序列化/反序列化

### 前端技术

- **HTML5**: 页面结构
- **CSS3**: 样式设计
- **JavaScript (ES6+)**: 交互逻辑
- **Tauri API**: 前后端通信

### 外部服务

- **OpenAI API**: AI 分析服务
- **Bilibili.com**: 内容源
- **Chromium**: 无头浏览器

## 扩展性设计

### 添加新的视频平台

1. 在 `src/` 下创建新的平台模块（如 `youtube.rs`）
2. 实现类似 `BilibiliClient` 的接口
3. 在 `commands.rs` 中添加相应命令
4. 在前端添加平台选择界面

### 支持更多 LLM 服务

1. 修改 `llm.rs` 中的 `LLMClient`
2. 添加新的 API 端点配置
3. 实现适配器模式支持不同服务
4. 在 `.env` 中添加配置选项

### 增强分析功能

1. 在 `models.rs` 中添加新的分析结果类型
2. 在 `llm.rs` 中添加新的分析方法
3. 在前端添加新的展示组件
4. 更新提示词模板
